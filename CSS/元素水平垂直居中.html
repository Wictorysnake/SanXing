# 元素水平垂直居中的方式有哪些

元素水平垂直居中的方式有哪些？

  • absolute + margin
  • fixed + margin
  • display: table
  • 行内元素 line-height
  • flex
  • transform 未知元素高度解决方案

# absolute + margin

.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 50%;
  left: 50%;
  margin: -50px 0 0 -50px;
}
